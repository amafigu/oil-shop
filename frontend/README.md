Frontend application created with create-react-app

Folders content:

/_mocks_: contains helper functions and data structure for test, as standard for jest.

/component: contains react functional components with correspondents styles sheets and tests.

All CRUD operations related components are in /ProductsCrud and /UsersCrud

/context: contains context providers for: language translation, current user and current purchased items.

/i18n: JSON files, one for each language.

/pages: contains all pages managed with react-router, the ones that are in a folder have related components within.

/styles: contains scss related folders and files, as mixis, typography helpers, breakpoints, etc.

/utils: contains files with helper functions.
